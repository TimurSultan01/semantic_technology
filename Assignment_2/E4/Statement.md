# Preparation

For preparation, we did the tutorial from the assignment sheet. A lot of information and tutorials on how to use Protégé can be found in the official Protégé Wiki. For a fundamental introduction to Protégé, this tutorial together with the more detailed documentation on the Protégé user interfaces was helpful.

We further looked in the literature and found a similar example from the Protégé paper (https://protege.stanford.edu/conference/2006/submissions/slides/OWLTutorial_Part1.pdf), namely the Pizza ontology example (https://protege.stanford.edu/ontologies/pizza/pizza.owl).

We used this example as best practice and designed our model analogous to it, as the pizza maker has similarities to the sandwich maker exercise.

We validated our model using the reasoner HermiT 1.4.3.456 and the OntoGraf graph view. This can be seen in the screenshots and in the reasoner log file. One of the images shows the inferred class hierarchy computed by the reasoner.
